    hex-to-rgb

    @function hex-to-rgb($color) { ... }

    Description

    Converts a hexadecimal color value to its RGB components

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $color

    El color hexadecimal a convertir

    		Color— none

    Returns

    List —

    Una lista que contiene los componentes rojo, verde y azul o NULL si el input no es valido

    Example

    $rgb: hex-to-rgb(#ff9900);
// $rgb: 255, 153, 0

    • Jesus
